Keep Your Projects Safe

Never lose your work. Follow these simple steps to backup and restore your projects using a single `.webide` file.

Backup Project

Save a full copy as a .webide file.

How to back up:

  • 1.Open the project you want to back up.
  • 2.Tap the menu icon (three dots) in the editor.
  • 3.Select 'Backup Project' from the options.
  • 4.Choose where to save your `.webide` backup file.
This saves your entire project, including all files and folders, into a single portable file.

Restore Project

Import from a .webide file.

How to restore:

  • 1.On the main project list screen, tap the '+' button.
  • 2.Choose 'Import from Backup'.
  • 3.Find and select your `.webide` backup file.
  • 4.The project will be added to your project list.
Restoring creates a new copy and will not overwrite existing projects with the same name.

Most internet users browse on smartphones, making mobile-first design essential. By building directly on your phone with Web IDE, you're developing and testing in the same environment as your audience. This gives you an inherent advantage in creating responsive, user-friendly experiences.

Web IDE provides a full-featured code editor, intelligent auto-completion, and a real-time live preview for immediate feedback. This accelerates your workflow and makes coding more intuitive. Plus, with offline capabilities, your productivity is never tied to an internet connection.

Use Your Favorite Tools

Web IDE is compatible with a wide range of popular front-end frameworks and libraries, giving you the freedom to build your way.

HTML & CSS Frameworks

Any framework that compiles to standard HTML and CSS is fully supported. Just link to a CDN or include the files in your project.

Tailwind CSS (CDN) Bootstrap Bulma Foundation

JavaScript Libraries

You can use any JS library or framework that runs directly in the browser. Simply include the script via a CDN link.

Libraries requiring a build step (like transpiling JSX) are not supported. For example, React needs compilation, but you can use browser-ready alternatives.

Vue.js jQuery Alpine.js Three.js React.js
